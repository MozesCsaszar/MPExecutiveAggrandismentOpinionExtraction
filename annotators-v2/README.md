# Annotators V2
- the second version of the annotators
- it brings real word-based labeling functions instead of using random ones
- most of these were generated by ChatGPT and all validated by me, no expert knowledge
- they run pretty poorly, with only a few PRO and CON results that can be found anywhere, but that's to be expected (30 and 60 of each from the 2014 and 2015 datasets, respectively)
- most stuff is categorized as neutral and probably with good reason
- it also contains some heuristics, which aren't implemented in the case of PRO and CON labeling
